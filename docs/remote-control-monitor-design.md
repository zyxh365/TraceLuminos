### 2.2 查询表单组件规范

| 组件 | 类型 | 占位符 | 验证规则 | 快捷操作 |
|------|------|--------|----------|----------|
| TraceId | 输入框 | 输入完整的TraceId | 16-64位十六进制 | 📋粘贴 🔍扫描 |
| VIN | 输入框 | 输入17位VIN码 | 17位字符 | 📋粘贴 🚗选择 |
| 时间范围 | 日期选择器 | 选择时间范围 | 开始≤结束 | ⏰快速选择 |
| 指令类型 | 下拉多选 | 全部 | 预定义列表 | 🔍搜索 |
| 链路状态 | 复选框组 | 全选 | 至少一个 | 全选/反选 |
| TBox状态 | 复选框组 | 全选 | - | 全选/反选 |

```
加载中 → 加载成功 → 数据展示
   ↓         ↓
加载失败 → 重试 / 空状态
```

```
输入查询条件 → 验证输入 → 发起查询
                      ↓
                 查询中 (显示loading)
                      ↓
              ┌───────┴───────┐
              ↓               ↓
          查询成功         查询失败
              ↓               ↓
          展示结果        显示错误/重试
              ↓
          用户交互 (点击/筛选/导出)
              ↓
          更新视图
```

```
启动监控 → 建立WebSocket → 接收数据 → 更新界面
              ↓               ↓
         连接失败          断开连接
              ↓               ↓
          自动重连        停止监控
```
